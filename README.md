# Assignment todo CLI

`todo` 命令是一个待办清单（to-do list）类的软件，用于帮助我们进行简单的任务管理。

### `init` command

用户在初次使用时，须先执行该命令以进行必要的初始化操作，如：创建保存数据的文件。

### `list` command

通过该命令来查看当前待办清单内的任务

### `add` command

通过该命令来向待办清单中添加新的待办任务

### `edit` command

通过该命令来修改已存在的待办任务

### `remove` command

通过该命令删除待办清单中的任务
